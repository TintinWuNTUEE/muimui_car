# Remote Control Simulation

## Prerequisites
```
$ pip3 install -r requirements.txt
```

## How to run

- To set up,
```bash
# create .env file and fill your SERVER_LINK
SERVER_LINK=http://localhost:8000/<API>

```
- To run in dev mode
```bash
$ ./muimui_app.sh -v
```
- To run in real mode
```bash
$ ./muimui_app.sh 
```
